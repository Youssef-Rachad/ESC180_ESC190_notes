Foreword

There are 2 dominant operating system types which present major differences in the command run and the programs used. First there is UNIX, which comprises of Apple and Linux machines. Then there is MSDOS, which comprises of Windows machines. (Note that Windows computers support UNIX commands through Powershell or by installing WSL Linux)

File System

UNIX machines and WSL

Open your terminal and follow along. There's a good chance you'll open onto the home directory (yourname/~). You can double check this by running pwd. From here let's make an ESC190 folder: mkdir esc190 Now let's navigate to it: cd esc190 and list the files inside using ls (It should return empty since there are no files in our folder yet)
We can create a file in a few ways
  • touch FILENAME creates an empty file
  • Use your text editor, such as textedit, gedit, vscode or vim
For this example, let's write an easy hello world: 
#include <stdio.h>
int main()
{
  printf("Hello ESC190!\n");
  return 0;
}
Once you've created a file and saved it, return to your terminal and run ls again. Ta-Da we can see our file in the directory. Take some time to familiarise yourself with these commands and search online for a deeper understanding.
